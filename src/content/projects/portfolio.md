This very site! Designed and built from scratch to showcase my projects and experience.
Key features include:

- **Zero-JS by default** — interactive components load only when needed via Astro Islands
- **Interactive terminal** — a Preact-based terminal widget with custom commands
- **Dark/light theme** — system-aware with manual toggle
- **Content Collections** — type-safe Markdown-driven project data
- **Lighthouse perfect scores** — optimized for performance, accessibility, and SEO
