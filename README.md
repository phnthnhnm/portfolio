# phanthanhnam.com

My personal portfolio — a fast, static site built with Astro and deployed on Cloudflare Pages.

**[phanthanhnam.com](https://phanthanhnam.com)**

## Tech stack

Astro · Tailwind CSS · TypeScript · Preact · Cloudflare Pages

For architecture, design decisions, and feature details, see the **[project page](https://phanthanhnam.com/projects/portfolio/)**.

## Running locally

```bash
pnpm install
pnpm dev        # http://localhost:4321
pnpm build      # outputs to dist/
pnpm test       # Vitest
```
